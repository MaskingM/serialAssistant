# serialAssistant
串口调试工具  

**开发原因：**硬件小哥测试硬件电路远程升级

**环境说明：**开发环境基于Qt5.8  
**功能说明：**  
1.发送16进制数据  
2.发送文件，发送文件时会进行更新进度条  

**代码说明：**
1.采用子线程进行接收数据  
2.采用主线程进行发送数据，发送文件时，由于无法放在子线程中进行，又考到，循环发送会造成 UI 卡死，所以使用定时器进行迂回发送
